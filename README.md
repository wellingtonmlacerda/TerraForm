TerraFormer 2D — Terraforming Prototype (Win32 + OpenGL)

Um protótipo jogável inspirado em “terraformação”: você constrói módulos de energia/água/oxigênio e usa um Beacon para transformar o terreno em algo mais “Terra”.

Build rápido (Windows):
- Abra um “Developer Command Prompt” (ou use o `build.bat`)
- `build.bat`

Controles:
- Movimento: `WASD` / `Setas`
- Pular/Nadar: `Space` (segurar para subir na água)
- Correr: `Shift`
- Minerar: segurar `LMB` (ou `E`)
- Coletar recursos: encoste nos drops (itens no chão)
- Colocar bloco/módulo: `RMB` (mostra preview)
- Seleção (hotbar): `1-6` (recursos) / `7-0` (módulos)
- Salvar: `F5`
- Carregar: `F9`
- Pausar: `Esc`
- Menu (pausado): `Q`

Objetivo:
- Aumentar `Energy / Water / Oxygen` com módulos.
- Colocar `Terraformer Beacon` e ir “esverdeando” o mapa até subir o indicador `Terraform` no HUD.

Notas:
- Sem assets externos: tudo gerado por código.
- Save: `save_slot0.tf2d` (na pasta onde o exe roda).
